# Using VS Code
> http://ruddra.com/2017/08/19/vs-code-for-python-development/

**vscode**에 `Django` 개발환경 설정하기.

```bash
$ pip install pylint_django
```
> `pyLint`를 설정하면서 `pylint_django`를 Arguments 로 던져주고 있다. 
> 해당 패키지를 설치해 주어야 vscode 에서 에러가 나지 않는다.
